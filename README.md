# penjelasan singkat :

1. Panic Button memungkinkan pengguna mengirimkan sinyal darurat secara instan hanya dengan satu sentuhan.

2. Dengan fitur Panic Button, bantuan dapat dipanggil dengan cepat saat terjadi kondisi darurat.

3. aplikasi ini bermanfaat sekali untuk rumah sakit dan masyarakat umum.

  # Fitur :
  1. Menggunakan smartphone
  2. GPS REAL TIME
  3. Whatsapp
     
   
Fitur Panic Button dirancang untuk memberikan respons cepat dalam situasi darurat dengan mengirimkan sinyal GPS secara otomatis ke pihak terkait.
Panic Button membantu meningkatkan keamanan pengguna dengan sistem peringatan darurat real-time.

Satu tombol, perlindungan maksimal. Panic Button siap digunakan kapan saja saat keadaan genting.
